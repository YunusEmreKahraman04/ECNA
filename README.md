# EV Charge Atlas

Elektrikli arac sarj istasyonlari icin gelistirilen web tabanli harita uygulamasi.

Bu README, guvenlik ve urun gizliligi sebebiyle minimum teknik bilgi icerir.

## Hizli Baslangic

```bash
npm install
npm run dev
```

## Uretim Build

```bash
npm run build
npm run preview
```

## Docker

```bash
docker compose up --build
```

Varsayilan erisim adresi:

- http://localhost:8080

## Not

Detayli mimari, veri akisi ve operasyonel dokumantasyon repo disinda yonetilmektedir.
